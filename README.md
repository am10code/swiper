# Swiper — расширение для задач и фокуса

Расширение для Chrome/Chromium с управлением задачами, карточкой фокуса и режимом ТаскСвайпер.

## Возможности

- Создание, редактирование и завершение задач, быстрые действия и контекстное меню.
- Категории и приоритеты (обычный/высокий), дедлайны с подсветкой.
- Разделы задач: просроченные, на сегодня, позже; режим отображения «все» или «только сегодня».
- Поиск по задачам и фильтрация по категории/приоритету.
- Полная карточка задачи: следующие шаги (drag & drop), лог, фокус‑таймер.
- Помодоро с историей сессий, итоговым временем и фоновым звуком.
- ТаскСвайпер: разбор очереди задач свайпами, undo и горячие клавиши.
- Раздел «Часто откладываемые» на основе счетчика переносов.
- Импорт/экспорт задач в `.swiper`/`.json`.
- Уведомления по дедлайнам и по завершению помодоро.

## Установка

1. Откройте Chrome и перейдите в `chrome://extensions/`.
2. Включите «Режим разработчика» (Developer mode).
3. Нажмите «Загрузить распакованное расширение» (Load unpacked).
4. Выберите папку с расширением.

## Точки входа

- `main.html` — основная вкладка расширения с разделами задач.
- `swiper.html` — отдельная страница ТаскСвайпер.
- `popup/popup.html` — компактный popup (исторический интерфейс).

## Архитектура

- `popup/storage.js` — работа с данными в `chrome.storage.local`.
- `popup/popup.js` — основная логика UI, разделов и списков задач.
- `popup/task-card.js` — полная карточка задачи и помодоро.
- `popup/modules/import-export.js` — импорт/экспорт задач.
- `popup/modules/swiper/*` — ТаскСвайпер и навигация.
- `background/background.js` — сервис‑воркер уведомлений и алармов.

## Структура проекта

```
swiper/
├── manifest.json                 # Конфигурация расширения (Manifest V3)
├── main.html                     # Основная вкладка расширения
├── swiper.html                   # Отдельная страница ТаскСвайпер
├── background/
│   └── background.js             # Service worker (уведомления/алармы)
├── popup/
│   ├── popup.html                # Компактный popup (legacy)
│   ├── popup.css                 # Стили для всех страниц
│   ├── popup.js                  # Логика UI и списков задач
│   ├── storage.js                # Хранилище и модели данных
│   ├── task-card.js              # Полная карточка задачи
│   └── modules/
│       ├── import-export.js      # Импорт/экспорт задач
│       └── swiper/
│           ├── swiper.js         # Логика ТаскСвайпер
│           ├── swiper-init.js    # Инициализация ТаскСвайпер
│           └── swiper-nav.js     # Навигация на swiper.html
├── assets/
│   ├── audio/                    # Звуки помодоро
│   └── icons/                    # Иконки расширения
├── STRUCTURE.md                  # Документация UI
└── README.md                     # Документация проекта
```
