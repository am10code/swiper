# Аудио для расширения

Для полной работы помодоро и уведомлений нужны файлы в этой папке:

- bell.wav          — звук завершения помодоро/напоминания
- pomodoro-ambience.mp3 — фоновый звук во время сессии помодоро

Код ссылается на эти пути в popup/task-card.js и background. Без файлов звуки не воспроизводятся; остальная функциональность сохраняется.
