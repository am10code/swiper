# Иконки расширения

Для работы расширения нужны PNG файлы размером 16x16, 48x48 и 128x128 пикселей.

SVG файлы уже созданы в этой папке. Конвертируйте их в PNG используя:
- Онлайн конвертер (например, https://cloudconvert.com/svg-to-png)
- ImageMagick: convert icon16.svg -resize 16x16 icon16.png
- Или запустите скрипт convert-icons.sh из корня проекта

После конвертации убедитесь что файлы называются:
- icon16.png
- icon48.png  
- icon128.png
